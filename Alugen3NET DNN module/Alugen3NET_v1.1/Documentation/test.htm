The Alugen3NET Module is usefull to manage product keys, customers and generated codes
based on ActiveLock for .NET from www.activelocksoftware.com .

Add Alugen3NET Product

  1. Add a Alugen3NET module, or go to an existing Alugen3NET module.
  2. Click Add new product.
  3. At Product Name, enter the product name.
  4. At Product version, enter version of this product.
  5. Press Generate new codes button to generate new pair of Vcode and Gcode.
  6. If something goes wrong press Clear codes button to clear existing codes.
  7. Alternately you can verify if existing codes are valid by pressing Validate button.
  8. Alternately, generated Vcode can be copied into clipboard by pressing Copy VCode button (icon).
  9. Alternately, generated Gcode can be copied into clipboard by pressing Copy GCode button (icon).
  10. Optional we can enter a price for this product.

Add Alugen3NET Customer

  1. Add a Alugen3NET module, or go to an existing Alugen3NET module.
  2. Click Add new customer.
  3. At Customer Name, enter the name of customer company or name.
  4. At Customer Adress enter the adress of this customer.
  5. At Contact Name enter the contact person name for this customer.
  6. At Customer phone enter the phone number contact of this customer.
  7. At Customer email enter the email adress of this customer. This email adress will be used in Edit License Code section to email activation code to customer.

Add Alugen3NET Code

  1. Add a Alugen3NET module, or go to an existing Alugen3NET module.
  2. Click Add new license code.
  3. At Product, select an existing product from the dropdown list.
  4. At Customer, select an existing customer from the dropdown list.
  5. At License type , select an existing licensing type from the dropdown list. Based on this selection thw folowing text boxes may appear.
    1. If license type is Time locked a text box named Exp Date will be visible. Here we must enter expiration date of generated code.
    2. If license type is Periodic a text box named Days will be visible. Here we must enter days of validity of generated code.
    3. If license type is Permanent there will be no other text box available.
  6. At Instalation code, enter instalation code received from customer in the activation process.
  7. Alternativelly if code is already in clipboard it can be pasted by pressing Paste Install Code button (icon).
  8. Press Generate button to generate activation code.
  9. Alternativelly, generated code can be copied into clipboard by pressing Copy Activation Code button (icon) or.
  10. Alternativelly, generated code can be emailed to selected customer by pressing Email Activation Code button (icon).
